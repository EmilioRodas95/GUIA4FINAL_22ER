Etiquetas Básicas de HTML

Definen la estructura general de una página web.

1. html

Inicio: html
Cierre: /html
Contenedor principal de una página web.

2. head

Inicio: head
Cierre: /head
Contiene información no visible del sitio.

3. title

Inicio: title
Cierre: /title
Define el título que aparece en la pestaña.

4. body

Inicio: body
Cierre: /body
Contiene todo el contenido visible

5. p

Inicio: p
Cierre: /p
Define un párrafo.

Etiquetas Básicas de para Tablas

Etiquetas que tienen como función el presentar información en forma de tabla en una página web.

Etiqueta Función
1. table Inicio: table
Cierre: /table
Crea una tabla en HTML.
2. tr Inicio: td
Cierre: /td
Crea una fila en una tabla.
3. td Inicio: tr
Cierre: /tr
Crea una celda de datos.
4. th Inicio: th
Cierre: /th
Crea una celda encabezando.

Etiquetas Básicas de Listas

Encargadas de ordenar informacíon por medio de diferentes recursos disponibles.

  1. ol
    • Inicio de la etiqueta: ol
    • Cierre de la etiqueta: /ol
    • Crea una lista numerada y ordenada.

  2. ul
    • Inicio de la etiqueta: ul
    • Cierre de la etiqueta: /ul
    • Crea una lista con viñetas.

  3. li
    • Inicio de la etiqueta: li
    • Cierre de la etiqueta: /li
    • Elemento de una lista.
